# Tic-Tac-Toe AI

## Objective
Develop an AI-powered Tic-Tac-Toe game where a human player competes against an intelligent computer opponent. The AI uses the Minimax algorithm to make the best possible moves, making it difficult to beat.

## Features
- Human vs AI gameplay
- AI implemented using the Minimax algorithm
- Interactive command-line interface
- Displays the game board after every move
- Detects wins, losses, and draws
- Prevents invalid moves

## Technologies Used
- Python 3
- Minimax Algorithm
- Math Module

## How to Run

1. Make sure Python is installed.
2. Open the terminal or command prompt.
3. Navigate to the project folder.
4. Run the following command:

```bash
python tic_tac_toe.py
```
