# Rule-Based Chatbot

## Objective
Develop a simple chatbot using Python that responds to user inputs based on predefined rules.

## Features
- Greets the user
- Answers common questions
- Displays current date and time
- Responds to thank you messages
- Ends the conversation with "bye"

## Technologies Used
- Python 3
- if-elif-else statements
- datetime module

## How to Run

```bash
python chatbot.py
```
