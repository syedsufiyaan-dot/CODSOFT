# Rule-Based Chatbot

## Objective

Develop a simple chatbot using Python that responds to user inputs based on predefined rules. The chatbot uses conditional statements to understand user queries and provide appropriate responses.

## Features

- Greets the user
- Responds to common questions
- Displays the current date
- Displays the current time
- Responds to "thank you" messages
- Ends the conversation when the user types "bye"
- Handles unknown inputs with a default response

## Technologies Used

- Python 3
- if-elif-else Statements
- datetime Module

## How to Run

1. Install Python 3 on your computer.
2. Open Command Prompt or Terminal.
3. Navigate to the project folder.
4. Run the following command:

```bash
python chatbot.py
```
